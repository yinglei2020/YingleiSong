This folder contains the code for the dynamic programming approach developed for the enhancement of color images. 
By G. Li, M. Rana, J. Sun, Y. Song and J. Qu

test.m is the file that contains the code for enhancing a single image, just replace the filename by the name of the image, execute it and 
the results of both the dynamic programming approach and HE will be displayed.

testall.m is the file that contains the code for performing the enhancement of BSD500 dataset. Change the folder name in the code,execute it, all images in
the folder will be enhanced and the analysis of the enhancement results would be performed.
